## Installasjon
#### Install all packages
```bash
npm install
```

#### Add fontawesome
```bash
npm config set "@fortawesome:registry" https://npm.fontawesome.com/
npm config set "//npm.fontawesome.com/:_authToken" F01127F1-8BB5-47F8-AE58-CBF0185F34C6
```

#### Add .env file with necessary keys
```bash
# General
VITE_API=
```

#### Run the application
```bash
npm run dev
```
